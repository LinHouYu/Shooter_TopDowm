# README - Juego 3D Top-Down Shooter

## Descripción del Proyecto
Este es un proyecto de un juego de disparos en 3D con perspectiva isométrica/cenital. En el juego, asumes el papel de un agente secreto que se ha infiltrado en territorio enemigo. 

La condición para ganar y terminar el nivel es explorar el mapa hasta encontrar el cofre del tesoro oculto. Ten en cuenta que los enemigos están programados no solo para atacarte, sino también para esquivar tus disparos, por lo que vas a tener que moverte de forma estratégica.

## Controles Básicos
* **W, A, S, D**: Movimiento del jugador.
* **Espacio**: Esquivar (Dodge).
* **R**: Recargar munición. (Nota: el juego tiene un sistema de recarga automática si te quedas sin balas en el cargador).
* **Click Izquierdo**: Disparar.

## Power-Ups (Objetos de mejora)
A lo largo del nivel puedes recoger distintos items que te van a dar ventajas en el combate:

* **Item Rojo (Salud)**: Restaura puntos de vida. Ideal para recuperarte después de un tiroteo cruzado.
* **Item Azul (Escudo)**: Te otorga invencibilidad temporal. Mientras el escudo esté activo, los ataques enemigos no te harán daño.
* **Item Morado (Agilidad)**: Es un buff completo de movilidad y ataque. Reduce el tiempo de espera para volver a esquivar, aumenta la cadencia de disparo y reduce el tiempo que tardas en recargar el arma.
